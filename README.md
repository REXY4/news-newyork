## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Build aplication : 
```
npm run build
or
yarn run buil
```
Running after build
```
npm run start
or 
npm start
```
